Takeaways

  • SITA wins on every percentile. Lowest p50, p90, and p99. No request above 70 ms across the run.
  • LWL has two ~1 s outliers on server 5001 (requests 25 and 29). LWL routes by Σ x, which doesn't know about the random Pareto multiplier; one straggler poisons a server and trailing small jobs queue behind it.
  • Random's load split (14 / 2 / 14) is wildly uneven, but at this light load the tail is still moderate - the imbalance only hurts more as λ rises.
  • This run is small and at low load. The interesting story will appear at higher λ (saturation) and lower α (heavier tails).
